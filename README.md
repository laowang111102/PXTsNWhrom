# 图书信息管理系统 python95

## 项目概述

图书信息管理系统是一款基于Python与PyQt5开发的软件，旨在提供便捷的图书数据管理功能。本项目包含用户界面设计（UI）、源代码以及打包后的可执行文件（exe）。系统支持两种数据库连接方式：SQLite和MySQL。

## 技术栈

- **开发语言**：Python
- **图形界面库**：PyQt5
- **数据库支持**：SQLite、MySQL

## 功能模块

### 图书管理

- 显示图书表中所有图书的相关信息
- 将图书信息插入到指定位置
- 将指定位置图书删除
- 实现图书信息去重

### 查询与修改

- 实现图书信息的模糊查询
- 批量修改图书信息

## 系统角色

- 系统管理员：负责图书信息的管理与维护
- 普通用户：查询和修改个人图书信息

## 运行环境

- 操作系统：兼容Windows、Linux、MacOS
- Python版本：3.6及以上

## 部署步骤

1. 安装Python环境
2. 安装PyQt5库
3. 安装SQLite或MySQL数据库
4. 运行可执行文件（exe）
